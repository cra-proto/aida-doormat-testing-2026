TPS/TVH pour les entreprises

Vous cherchez peut-être : Paiements du crédit pour la TPS/TVH pour les particuliers

Obtenez votre code d’accès dans votre compte de l’ARC

À compter du 20 octobre 2025, les propriétaires d’entreprise pourront voir leur code d’accès pour la TPS/TVH dans leur compte de l’ARC (dans Mon dossier d’entreprise seulement).

Découvrez comment obtenir un code d’accès.

Les particuliers dont le NAS temporaire commence par 9 doivent s'inscrire en utilisant l'IDE

Si votre numéro d'assurance sociale (NAS) temporaire commence par 9, vous devez utiliser l'Inscription en direct des entreprises (IDE) pour vous inscrire et obtenir instantanément un numéro d'entreprise (NE) ainsi que des comptes de programme de l'ARC. Pour en savoir plus, consultez Renseignements importants pour les particuliers dont le NAS temporaire commence par 9.
